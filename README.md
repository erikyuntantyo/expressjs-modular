# expressjs-modular

> Implementation ExpressJS modular approach with ES6

## About

This is the a ExpressJS modular approach example that uses ES6.

## Getting Started

1. Make sure you have [NodeJS](https://nodejs.org) and [npm](https://www.npmjs.com) or [yarn](https://yarnpkg.com) installed. And also [nodemon](https://nodemon.io/) to monitor the source and automatically restart the server when any changes in the source.
2. Install dependencies
    ```$ cd path/to/expressjs-modular```

    Using `yarn`
    ```bash
    $ yarn
    ```

    Using `npm`
    ```bash
    $ npm install
    ```

3. Run
    Using yarn
    ```bash
    $ yarn start
    ```

    Using npm
    ```bash
    $ npm start
    ```

4. Test
    Using yarn
    ```
    $ yarn test
    ```

    Using npm
    ```
    $ npm test
    ```
